# TASK 2 — Scanner Pair Selection Engine (Medium)

## Goal

Watchlist should intelligently choose the correct trading pair.

---

## Current Problem

User adds:

```text id="tyl2nq"
BTC
LINK
ETH
```

System assumes:

```text id="tw7yz5"
COIN/INR
```

even if INR pair does not exist.

---

## Desired Behaviour

Priority:

```text id="w7tk6m"
1. INR pair
2. USDT pair
3. Reject if neither exists
```

Examples:

```text id="6swg9l"
BTC -> B-BTC_INR
ETH -> B-ETH_INR
PEPE -> B-PEPE_USDT
```

---

## Implementation

Create helper:

```python id="q8xg6e"

```

Returns:

```python id="aj8e1c"

    "coin": "BTC",
    "pair": "B-BTC_INR",
    "quote": "INR"
}
```
